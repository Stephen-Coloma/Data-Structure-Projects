## Development Guidelines

### Tasks Breakdown

#### Task 1: Infix to Postfix Converter
- **Sub-Task 1**: Parse and validate the infix expression.
- **Sub-Task 2**: Implement the infix to postfix algorithm.
- **Sub-Task 3**: Display the conversion in the specified format.

#### Task 2: Postfix Expression Evaluator
- **Sub-Task 1**: Parse and validate the postfix expression.
- **Sub-Task 2**: Implement the postfix evaluation algorithm.
- **Sub-Task 3**: Display the evaluation process and result.
